Anyone can have a green thumb.

myGarden is a webApp created to give a user a virtual representation of their physical garden. myGarden consolidates helpful planting and harvesting information to one location. In addition to providing information, myGarden automates alerts for both planting times and watering frequencies by using manually logged water dates as well as rain data from Weather Underground

Group Projects - Front-End

From my first Pull Request to my last sprint, group projects were an imperative part of my learning during my time at Nashville Software School. Below are a couple of these.

Theme Park

Theme Park's objective was to create a interactive map of a theme with 8 areas with different attractions and events. The required technologies were SASS, JQuery, Grunt (with JSHINT), Promises, Browserify,Handlebars, Grid framework, Firebase for data storage and retrieval.

  • Date: January 2018
  • See Github Repository

Pinterest

Our goal with this project was to reverse engineer the basic functionality of Pinterest and build our own version of it. To do this, we used http verbs POST, GET, PUT, and DELETE to support CRUD functionality for authenticated users. This app also required use of AngularJS framework.

  • Date: February 2018
  • See Github Repository

Group Projects - Back-End

From my first Pull Request to my last sprint, group projects were an imperative part of my learning during my time at Nashville Software School. Below are a couple of these.

Bangazon API

Building Bangazon INC's API is the first sprint in our back-end projects. Prior to beginning, our group decided on the finalized ERD based on project requirements. We then created data and populated the tables with that data for querying. This API will allow user to GET/POST/PUT and (sometimes) DELETE items from the Bangazon Database. This will use express routing and sqlite.

  • Date: March 2018
  • See Github Repository
